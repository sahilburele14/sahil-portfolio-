# Sahil Burele — Portfolio Website

A futuristic, responsive portfolio website built with pure HTML, CSS, and JavaScript.

## 🚀 Live Demo
> Deploy via GitHub Pages — link will appear after deployment

## ✨ Features
- Animated splash screen with monogram
- Particle + gradient mesh background
- Scroll-based reveal animations
- Experience accordion timeline
- Animated achievement counters
- Dark / Light theme toggle
- Mobile responsive with burger menu
- Scroll progress bar & section dot navigation

## 🛠️ Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Canvas API (animated background)
- Google Fonts (Syne, JetBrains Mono, DM Sans)

## 📦 How to Run Locally
Just open `index.html` in your browser — no install needed.

## 🌐 Deploy on GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set Source to `main` branch → `/ (root)`
4. Click **Save**
5. Your site will be live at `https://yourusername.github.io/sahil-portfolio/`
